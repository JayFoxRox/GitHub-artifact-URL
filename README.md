# GitHub-artifact-URL

A small Python 3 HTTP server to return the latest GitHub Actions artifact URL given the following conditions:

- Owner name
- Repository name
- Branch name
- Workflow name
- Workflow event
- Artifact name

*(Only successful workflows will be considered!)*
